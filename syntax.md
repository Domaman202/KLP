```
namespace std
    // какой-то код

```

# Переменные

Константа `pi`
```
val pi: f32 = 3.14
```
Переменная `i`
```
var i: i32 = 12
```

# Функции

```
fun add(a: i32, b: i32): i32 = a + b
```
```
fun add(a: i32, b: i32): i32 = { a + b }
```
```
fun add(a: i32, b: i32): i32 = { return a + b }
```

# If-Else

```
if i > 5 then
    i += 1
else
    i -= 1
end
```

# Тернарные операторы

```
i = if i > 5 then i + 1 else i - 1
```
